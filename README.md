# musik_webshop

En modern, responsiv webshop för gitarrer, trummor och piano.

## Gruppmedlemmar
- Benjamin Simmons
- Leo Spenner

## Teknikstack
- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript (ES6+)
- localStorage för varukorgen

## Så här startar du projektet
1. Klona repot
2. Öppna index.html i webbläsaren (ingen server behövs)

## Funktioner som är klara
- Responsiv navbar med kategorier
- Produktlista med filtrering
- Varukorg med localStorage
- Enskild produktsida
